Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an element in the DOM =>

1) Using getElementById().

2) Using getElementsByClassName().

3)Using getElementsByTagName().

4) Using querySelector().

5) Using querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent =>

innerHTML => Set the HTML content inside an element , including tags, text and spacing .

innerText => Set the text inside an element, but ignore HTML tags and extra spaces . Its does not return hidden text , only return visible text .

textContent => Set the text content inside an element, including hidden text , but ignore HTML formatting.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM =>

Event delegation is a way to handle events by adding a listener to a parent element instead of
each child element . When a child triggers an event , the parent detects and manages it.
Simply put , the parent element controls its child elements events.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM =>

Event bubbling is called event propagation in the DOM.
When a child element triggers an event,it spreads from the parent element up to the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

Create, add, and remove elements using JavaScript =>

Creat element => Using document.createElement().

Add element => Using parentElement.appendChild().

Remove element Using remove() & removeChild().
